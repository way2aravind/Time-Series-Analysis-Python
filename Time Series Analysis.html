Let’s use the pandas library to load a time-indexed dataset and visualize it with Matplotlib. For this example, we’ll use a sample dataset containing monthly airline passengers.¶

In [1]:
# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Load a sample time series dataset
url = 'https://raw.githubusercontent.com/jbrownlee/Datasets/master/airline-passengers.csv'
data = pd.read_csv(url, parse_dates=['Month'], index_col='Month')

# Display the first few rows
data.head()

This dataset has two columns: the Month (our time index) and the Number of Passengers.¶

Visualizing the Data¶

We’ll plot the data to observe trends and seasonality.

In [2]:
# Plot the time series data
plt.figure(figsize=(12, 6))
plt.plot(data, label='Passengers')
plt.title('Monthly Number of Airline Passengers')
plt.xlabel('Date')
plt.ylabel('Number of Passengers')
plt.legend()
plt.show()

Using statsmodels, we’ll break down our time series data into its components.¶

In [3]:
from statsmodels.tsa.seasonal import seasonal_decompose

# Decompose the time series data
decomposition = seasonal_decompose(data['Passengers'], model='multiplicative', period=12)

# Plot the decomposed components
plt.figure(figsize=(10, 8))
plt.subplot(411)
plt.plot(data, label='Original')
plt.legend(loc='upper left')

plt.subplot(412)
plt.plot(decomposition.trend, label='Trend')
plt.legend(loc='upper left')

plt.subplot(413)
plt.plot(decomposition.seasonal, label='Seasonality')
plt.legend(loc='upper left')

plt.subplot(414)
plt.plot(decomposition.resid, label='Residuals')
plt.legend(loc='upper left')

plt.tight_layout()
plt.show()
